# 恶魔轮盘 - 弹药统计器
---

## ✨ 功能特性

### 🎮 游戏设置
- 灵活配置：支持自定义总子弹数量和实弹/空包弹分配
- 数据校验：自动确保输入数据的有效性，避免配置错误
- 一键重置：快速清空当前数据，开启新一轮游戏追踪

### 📊 实时追踪
- 子弹状态可视化：直观显示弹舱中每一发子弹的状态
  - 🔴 红色：实弹
  - ⚪ 灰色：空包弹
- 实时统计：显示剩余实弹、空包弹和总子弹数量
- 发射记录：快速标记并记录射出的子弹类型，数据自动更新
- 逆转器适配：独立记录逆转器发射的子弹，支持状态反转场景

### 📈 概率分析与推理辅助
- 剩余数量：实时展示当前剩余子弹中实弹/空包弹的数量
- 弹夹状态推测：可单独标记每一发子弹的状态，辅助逻辑推导
- 动态更新：根据已发射记录实时调整推测结果，降低决策难度

### 🔧 道具场景适配
- 逆转器模式：专门适配逆转器带来的子弹状态反转，避免计数混乱
- 发射历史回溯：完整记录本局所有发射记录，支持复盘操作
